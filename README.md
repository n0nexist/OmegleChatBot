# OmegleChatBot
A free omegle chat bot written in python3.

# How to install and run
```
# INSTALLATION
git clone https://github.com/n0nexist/OmegleChatBot
cd OmegleChatBot
bash install.sh
# RUN
cd src
python3 main.py
```
